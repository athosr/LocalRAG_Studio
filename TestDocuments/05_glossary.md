# Aurora Field Lab — glossary (demo)

| Term | Meaning |
|------|---------|
| **Island mode** | Off-grid operation using batteries and micro-hydro; sustained draw limit **18 kW**. |
| **Grid-tie** | Seasonal grid connection (June–August). |
| **LabOS** | Internal software suite; calibration wizard is in **v3.6** as of this document set. |
| **W-7** | Meltwater collection system feeding potable processing. |
| **E-77** | Sensor fault indicating desiccant replacement (**DP-5**) is required before calibration. |
